# Code of Conduct

## Our pledge

We aim to keep collaboration respectful, inclusive, and productive.

## Expected behavior

- Be respectful in reviews and discussions.
- Focus on the code and the problem.
- Give constructive feedback.
- Avoid harassment, discrimination, or personal attacks.

## Unacceptable behavior

- Harassment or intimidation
- Hate speech or discriminatory language
- Deliberate disruption of discussions
- Publishing private or sensitive information

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, and related communication.

## Reporting

If you need to report a violation, contact the project maintainers through the repository's issue tracker or the channel provided by the project owner.
